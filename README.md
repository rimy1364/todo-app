# Rimjhim's List

A clean and simple Todo List web application built with HTML, CSS, and JavaScript.

## Live Demo

[View Live on Vercel](https://todo-app-ten-lac-22.vercel.app)

## Features

- Add new tasks
- Mark tasks as done (click on task)
- Delete tasks
- Tasks are saved in browser (localStorage) — they persist after refresh
- Press **Enter** to quickly add a task

## Tech Stack

- HTML
- CSS
- JavaScript (Vanilla)

## Project Structure

```
todo-app/
├── index.html   # Main HTML structure
├── style.css    # Styling
└── app.js       # App logic
```

## Getting Started

To run locally, just open `index.html` in your browser — no setup needed.

```bash
git clone git@github.com:rimy1364/todo-app.git
cd todo-app
# Open index.html in your browser
```
